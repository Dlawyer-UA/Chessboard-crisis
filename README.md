# ♟ Шахова дошка криз

> Авторська методологія діагностики кризових проявів аграрних бізнес-організацій  
> **Козловський Д.О.** · Дисертаційне дослідження · Київ, 2025

---

## Що це

Веб-застосунок для оцінки кризового стану підприємств за методом **«Шахова дошка криз»**.  
Дозволяє:
- вводити функціональну спроможність Sj по 6 сферах;
- автоматично розраховувати ІКПj та системний індекс кризовості Ksystem;
- будувати теплову матрицю взаємовпливів Kij;
- порівнювати організації через ранжування та радарні діаграми;
- редагувати матрицю A під власні умови;
- зберігати дані між сесіями (localStorage).

## Формули

| Позначення | Формула | Діапазон |
|---|---|---|
| **Sj** | Вводиться вручну | [0; 10] |
| **ІКПj** | `1 − Sj / 10` | [0; 1] |
| **Kij** | `ІКПi × Aij × 10` | [0; 10] |
| **Ksystem** | `Σ(Wj × ІКПj)` | [0; 1] |

### Ваги сфер Wj
| Кадрова | Орг-упр | Операційна | Логістична | Фінансова | Стратегічна |
|---|---|---|---|---|---|
| 0.12 | 0.14 | 0.20 | 0.16 | 0.22 | 0.16 |

### Шкала ІКП
| Значення | Рівень |
|---|---|
| 0.00–0.20 | ✅ Стабільний |
| 0.21–0.40 | 🟡 Слабка криза |
| 0.41–0.60 | 🟠 Помірна криза |
| 0.61–0.80 | 🔴 Розвинена криза |
| 0.81–1.00 | 🚨 Глибока криза |

## Деплой на Vercel

### Варіант 1 — через GitHub (рекомендовано)

1. Завантажте код на GitHub:
```bash
git init
git add .
git commit -m "init: chess crisis board"
git remote add origin https://github.com/ВАШ_ЮЗЕР/chess-crisis-board.git
git push -u origin main
```

2. Зайдіть на [vercel.com](https://vercel.com) → **Add New Project**
3. Оберіть репозиторій → **Deploy**
4. Vercel автоматично визначить Vite. Жодних додаткових налаштувань не потрібно.

Через ~2 хв отримаєте URL: `https://chess-crisis-board.vercel.app`

### Варіант 2 — через Vercel CLI

```bash
npm i -g vercel
vercel login
vercel --prod
```

### Налаштування Vercel (вже є у vercel.json)

```json
{ "rewrites": [{ "source": "/(.*)", "destination": "/" }] }
```

## Додавання нової організації

1. Перейдіть на вкладку **Вхідні дані**
2. Введіть назву у поле → натисніть **Додати** або `Enter`
3. Налаштуйте слайдери Sj
4. Шахова дошка та ранжування оновляться автоматично
